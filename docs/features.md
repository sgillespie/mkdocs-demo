# Features

MkDocs is a static site generator for markdown.

 * Open Source ([BSD 2-clause](https://opensource.org/license/bsd-2-clause))
 * Themes
     * MkDocs
     * Read the Docs
     * and many more third-party themes
 * Built-in Dev Server
 * Customization
     * Themes
     * Markdown Extensions
     * Configuration
 * Easy deployment
     * GitHub Pages
     * Read the Docs
     * Backstage Developer Portal
     * Or any static HTML server
